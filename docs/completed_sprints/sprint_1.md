# Completed Sprint №1: Базовая инфраструктура и Создание упражнения (US-1)

**Цель спринта:** Настроить базовые слои приложения и полностью реализовать пользовательский сценарий создания кастомного упражнения (US-1).

## Статус: Успешно завершен

### Выполненные задачи:

1. **DevOps-1: Развертывание локального окружения СУБД**
   * **Статус:** DONE
   * **Описание:** Настройка Docker Compose с контейнером PostgreSQL 16.

2. **Backend-1: Инициализация сервера, конфигурации и инфраструктуры БД**
   * **Статус:** DONE
   * **Описание:** Каркас Go-приложения, подключение к БД (pgxpool) с ретраями, автомиграции.

3. **Backend-2: API Handlers**
   * **Статус:** DONE
   * **Описание:** Эндпоинты `POST /api/exercises` и `GET /api/exercises` с обработкой ошибок и авторизацией.

4. **Frontend-1: API Client**
   * **Статус:** DONE
   * **Описание:** Модуль API Client на Vanilla JS с отправкой запросов и X-User-Id заголовком.

5. **Frontend-2: State Management (Store)**
   * **Статус:** DONE
   * **Описание:** Реализация Pub-Sub Store и синхронизация списка упражнений с API.

6. **Frontend-3: UI Component (Challenge Form)**
   * **Статус:** DONE
   * **Описание:** Форма создания челленджа с динамическим полем для новых упражнений.

7. **QA-1: API Тестирование (Postman / cURL)**
   * **Статус:** DONE
   * **Описание:** Тесты позитивных и негативных (пустые строки, дубликаты) сценариев API упражнений.

8. **QA-2: UI/UX Тестирование (Браузер)**
   * **Статус:** DONE
   * **Описание:** Тестирование интерактивного скрытия/показа полей, успешных отправок и тостов.
