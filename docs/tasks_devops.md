# DevOps Tasks

## Epic: US-1 Создание упражнения

### Задача 1: Развертывание локального окружения СУБД (DevOps-1)
* **Файлы:** `docker-compose.yml`, `.env.example`
* **Описание:** Настроить и запустить Docker Compose с контейнером PostgreSQL 16. Предусмотреть persistent storage (volumes) для сохранения данных бэкаут-трекера и пробросить порт 5432 наружу. Подготовить шаблон `.env.example`.
* **Ограничения:** Изолировать сеть контейнера, настроить healthcheck. Задача является блокирующей для Backend-1.
